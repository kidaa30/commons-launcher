1. INTRODUCTION

The Launcher Component is designed to be a cross platform Java application launcher.

This code was originally developped as part of the Tomcat project.

2. DEPENDENCIES

The Launcher component is dependent upon the following external components for development and use:

  • Java Development Kit (Version 1.2 or later)
  • JUnit Testing Framework (Version 3.7 or later) - for unit tests only, not required for deployment

3. RELEASE INFO

Current Release: Launcher is yet to be released. We hope it will be RSN.
